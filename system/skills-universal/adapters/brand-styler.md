### 색상 사용 원칙

- **Primary = 브랜드 아이덴티티** — Lee Navy + Black + White가 기본 축. 특히 Navy는 브랜드 라벨/기획서 시그니처
- **Neutral Basic = 전 시즌 공용** — Cream/Ivory/Stone Grey 계열로 빈티지·헤리티지 감성 유지
- **Accent = 시즌 포인트** — 시즌마다 변경 (현재 시즌 `visual-identity.json` 확인)
- 배경이 Navy면 White 또는 Cream 텍스트, 포인트는 Accent 컬러
- 배경이 Cream이면 Navy 또는 Black 텍스트, 포인트는 Accent 컬러
- 채도 높은 컬러 남용 금지 — 2026 컬러 키워드는 "미들톤 보완·쿨 파스텔·그레이 확장·모던하고 세련된 무드"

## 그래픽 스타일

### 로고·심볼 운영 원칙

- **메인 로고:** Twitch Logo (필기체). 1번 표시 형태가 기본. 자수 10cm 이하는 ® 제거, 프린트는 모두 ® 포함
- **메인 심볼:** Buddy Lee (1920년대 Lee 광고 캐릭터 인형 — 전신/헤드)
- **서브 로고:** Vintage Logo 6종, House 지붕 박스 (Kansas City / Work Clothes / Since 1889 / Premium Quality)
- **House 로고 주의:** 워크웨어 관련 그래픽 제외 사용 지양 (카우보이·Buddy Lee 등 워크웨어 맥락과 병용)
- **아카이브:** Lee 101, Buddy Lee 오리지널 타이포, Storm Rider (카우보이/말 일러스트), Lazy-S 스티치, Union-Alls 계보 (필기체 Union Logo)
- **로고 변형 금지:** 형태 변형은 디자이너 재량이지만 컨펌 필수. 로고 자체의 본 형태 왜곡/재구성 금지

### 그래픽 제작 원칙 (Do)

- 어패럴·용품 적용 시 난해하지 않고 깔끔·심플한 그래픽
- 세련되고 현대적인 컬러 조합
- 메인 BI(트위치·빈티지·하우스·유니온) 활용한 가독성 높은 브랜드 이미지
- 성별 무관 공감 가능한 모티브의 트렌디 아트웍
- 아카이브(Lee 101·Storm Rider·Lazy-S·Union-Alls·Buddy Lee)를 트렌드에 맞게 재해석

### 그래픽 제작 금지 (Don't)

- 옷 위에 직접 그린 것처럼 보이는 그래픽 배제
- 과도한 정밀 묘사 / 과하게 카툰·귀여운 그래픽 지양
- 칙칙하고 매력 없는 컬러 구성 금지
- 구시대적·올드한 느낌의 아트웍 배제
- 3B(Beach/Bed/Baby) 착장 금지 (전사 가이드라인)

### 사진·비주얼 방향

- **메인 무드:** 레드 배경 + 베레모·데님 룩 시리즈 (Denim 스튜디오)
- **서브 무드:** 네이비 배경 + Casual 라이프스타일
- **로케이션:** 스트릿 · 자연광 혼합 (스튜디오 단독보다는 자연광 + 공간감)
- **뮤즈 2026:** NCT 재민 (남성·팬덤 유입) / 노윤서 (여성·남성 확장)
- 정적 포즈보다 움직임·분위기 있는 포즈 지향

### 레이아웃 원칙

| 원칙 | 설명 |
|------|------|
| Grid-based | 빈티지 카탈로그 감성의 단단한 그리드 |
| Heritage Framing | 박스 프레임·라벨 프레임 활용 (House 로고 지붕 박스 등) |
| Balanced White Space | 여백으로 프리미엄·모던 무드 확보 |
| Archive Layering | 아카이브 로고/심볼을 배경·워터마크처럼 레이어 |

  배경: Lee Navy 또는 Stone Grey

  배경: Lee Navy
  텍스트: White / Cream, Lee Twitch

## 품질 체크리스트

산출물에 Lee 브랜드를 적용한 후 확인:

- [ ] Primary 컬러 Lee Navy / Black / White 사용
- [ ] Cream · Stone Grey · Deep Grey 등 중간톤으로 헤리티지 감성 유지
- [ ] 시즌 Accent(26SS: Pink Yarrow / Quince / Tile Blue / Silver) 남용 없음 (포인트 요소에만)
- [ ] 헤딩에 Bold serif/slab 또는 Lee Twitch 계열 폰트
- [ ] 본문에 Clean sans-serif (Pretendard, Helvetica Neue 등)
- [ ] 빈티지 감성 유지 — 너무 플랫/디지털하면 Lee 감성 옅어짐
- [ ] 아카이브 모티프(Buddy Lee / 101 / Storm Rider / Lazy-S / Union-Alls) 활용 시 ip-bible.json 준수
- [ ] 로고 변형 금지 원칙 준수 (House 로고 워크웨어 맥락 외 사용 지양)
- [ ] 3B(Beach/Bed/Baby) 착장 금지
- [ ] 레이아웃 그리드·프레임 기반, 여백으로 프리미엄감 확보

## 주의사항

- 이 가이드는 **리(Lee) 전용**입니다. 다른 브랜드 출력물에는 사용하지 마십시오.
- HEX는 PANTONE 근사치이며, 인쇄·라벨 등 정밀 작업 시 PANTONE 기준 우선
- 시즌별 Accent 컬러는 `presets/lee/visual-identity.json` 최신 값 확인
- 고객 대면 콘텐츠는 반드시 `presets/lee/tone-manner.json`도 함께 참조
- 헤리티지 아카이브(Buddy Lee · Lee 101 · Storm Rider · Lazy-S · Union-Alls) 활용 시 `presets/lee/ip-bible.json` 참조
- 라이선서는 HD Lee Company (Kontoor Brands 산하) — 글로벌 로고/상표 가이드라인과 충돌 금지
